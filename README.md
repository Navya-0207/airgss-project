# AI-Powered Rural Governance Support System (AIRGSS)

## Description
AIRGSS is a web-based AI-powered governance platform designed to improve rural administrative services. It provides grievance management, chatbot interaction, scheme recommendations, and digital services.

## Features
- AI-based grievance classification (NLP)
- Multilingual chatbot (text & voice)
- Scheme recommendation system
- Digital payments and records
- Admin dashboard for monitoring

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
